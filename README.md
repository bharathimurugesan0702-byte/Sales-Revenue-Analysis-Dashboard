# Sales-Revenue-Analysis-Dashboard
A data analytics project that visualizes sales trends, category performance, top-selling products, and regional sales insights using Power BI.

## Objective
To analyze sales data and create an interactive Power BI dashboard for business insights.

## Tools Used
- Power BI
- CSV Dataset

## Dashboard Features
- Total Sales KPI
- Monthly Sales Trend
- Sales by Category
- Top Products by Sales
- Region Filter (Slicer)

## Key Insights
- Total sales reached approximately 2.26 million.
- Sales performance varied across categories.
- Top-performing products were identified.
- Region-based filtering allows focused analysis.
